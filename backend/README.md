## Setup and Installation
<br>

Install [Node Package Manager(npm)](https://sourceforge.net/projects/npm.mirror/)

Install [Node.js](https://nodejs.org/en/download)


<br>

> Initialize a npm project
```
  npm init
```

> Install The Express package
```
  npm install express
```

> Run a Node.js File
```
  node <file-name>
```
